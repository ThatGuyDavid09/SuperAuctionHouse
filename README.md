# SuperAuctionHouse
This is an Auction House plugin written for Minecraft spigot multiplayer.

## Features:
* ###Complete: 
    * Selling items at a fixed price
    * Infinite sell items (only admins)
    * Buy items
    * Vault integration
    * SQL backup integration
* ### In progress:
    * View other people's auctions
* ### Coming soon:
    * View your current bids
    * View your current auction
    * Timed auctions
    * H2 backup support
